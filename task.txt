На складе лежат посылки разного типоразмера, для простоты мы запишем это простыми целыми числами - юнит. Таким образом каждая посылка занимает некоторое, отличное от нуля, количество юнитов.

Приезжает грузовик, который может забрать только часть посылок из-за ограничений по кол-ву юнитов. Вам необходимо написать алгоритм расчета какие две посылки может забрать грузовик. Дополнительные условия:

- грузовик заберет ровно две посылки
- грузовик должен иметь резерв места в 30 юнитов
- грузовик должен посылками максимально заполнить свободное место

На вход функции будут подаваться размер грузовика и массив посылок.

int[] whichPackagesToLoad(int truckSpace, int[] packages) {
  // your code here
}

Например, вызов:
whichPackagesToLoad( 110, new int[] { 20, 70, 90, 30, 60, 110 } );
вернет:
20 и 60
